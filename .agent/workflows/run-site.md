---
description: How to run the White Wolf Furniture website locally
---

To run the project:

1. Open your terminal.
2. Navigate to the project directory:
```bash
cd /Users/apple/.gemini/antigravity/scratch/white-wolf-furniture
```
// turbo
3. Start the development server:
```bash
npm run dev
```
4. Open your browser and navigate to `http://localhost:4321`.
